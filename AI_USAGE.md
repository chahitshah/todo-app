# 🤖 AI Usage & Prompt Documentation

## 📌 Overview

As per the assignment guidelines, AI (ChatGPT) was used as a **supporting tool** during the development of this fullstack Todo application.

The primary objective was to **learn and implement all features independently**, while using AI only for:

- Understanding concepts  
- Resolving development blockers  
- Improving code structure and UI  

All major features were implemented and tested manually to ensure a clear understanding of the system.

Overall, AI contributed approximately **20–30% assistance**, mainly in guidance and debugging, while the majority of the implementation was completed independently.

---

## 🛠️ AI Tool Used

- ChatGPT (used for guidance, debugging, and improvement suggestions)

---

## 🧠 Development Approach

The project was developed in a structured and step-by-step manner:

1. Understanding assignment requirements  
2. Planning project architecture and folder structure  
3. Building frontend components using React  
4. Developing backend with Node.js and Express  
5. Integrating MongoDB Atlas database  
6. Implementing authentication (JWT + bcrypt)  
7. Connecting frontend and backend APIs  
8. Enhancing features and improving UI/UX  
9. Debugging and optimizing performance  

AI was used selectively during:

- Concept clarification  
- Debugging errors  
- UI/UX improvement suggestions

## 🎨 UI Design Approach

The UI design was inspired by modern SaaS dashboards and minimal productivity applications.

Focus areas:

- Clean and minimal layout  
- Consistent spacing and alignment  
- Light and Dark theme support  
- Improved readability and usability  

AI was used to:

- Translate design ideas into CSS structure  
- Suggest layout improvements  

Each design decision and final implementation was manually reviewed and refined.
